08 — البداية السريعة

🚀 كيف تستأنف العمل

خطوات 6 لتشغيل المشروع من جديد.

# 1. استنساخ المشروع git clone https://github.com/<your-org>/meditrans-core.git cd meditrans-core # 2. تثبيت التبعيات npm install # 3. إعداد متغيرات البيئة cp .env.example .env # عدِّل DATABASE_URL ليشير لـ PostgreSQL الخاص بك # 4. تطبيق migration (إنشاء الجداول) npm run db:migrate # 5. ملء بيانات البذر (كتالوج + أسعار + admin) npm run db:seed # 6. تشغيل سيرفر التطوير npm run dev → http://localhost:3000
