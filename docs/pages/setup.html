02 — الإعداد التقني

Stack المشروع 🔧

التقنيات المختارة لبناء MediTrans — كلها معروفة عندك كـ full-stack engineer.

Framework Next.js 14 App Router + Server Actions
Language TypeScript strict strict mode مفعَّل كاملاً
Styling Tailwind CSS tokens.css + tailwind.config
Database ORM Prisma PostgreSQL 15 · Port 5432
Validation Zod schema لكل module
Code Quality ESLint + Prettier LF line endings · @/* alias

🛠️ أوامر npm المتاحة

# تشغيل التطوير npm run dev → http://localhost:3000 # قاعدة البيانات npm run db:migrate → تطبيق migrations جديدة npm run db:seed → ملء بيانات البذر npm run db:studio → Prisma Studio (واجهة مرئية) npm run db:reset → إعادة تعيين كاملة ⚠️ # جودة الكود npm run lint → ESLint npm run format → Prettier
