03 — قاعدة البيانات

PostgreSQL — 11 جدول 🗄️

كل قاعدة البيانات مبنية وطُبِّقت عبر migration واحدة. فيما يلي شرح لكل جدول ولماذا موجود.

الجدولما فيهمثال عملي
users المستخدمون admin و assistant — اللي يدخلو على النظام
clients العملاء شركة تأمين، مستشفى، فرد — من يدفع الفاتورة
patients المرضى المريض اللي تم نقله أو معالجته
service_catalog كتالوج الخدمات "Transport urgent"، "ECG"، "Suture" — قائمة كل الخدمات الممكنة
pricing_rules قواعد التسعير Transport simple عادي = 150، urgence = 250 درهم
pricing_modifiers معدِّلات الأسعار إضافة الليل (+100 MAD) — اختيار يدوي فقط، لا كشف تلقائي
pricing_distance_rates أسعار المسافة 7.50 درهم لكل كيلومتر
services الخدمات المنفَّذة سجل كل خدمة فعلية تمت (نقل، حقنة...)
pricing_snapshots لقطات الأسعار سعر مجمَّد عند إنشاء الخدمة — لا يتغير أبداً
invoices + lines الفواتير وأسطرها فاتورة تجمع عدة خدمات في وثيقة واحدة
payments المدفوعات تسجيل الدفعات على كل فاتورة
system_config إعدادات النظام TVA 10%، ساعة بداية الليل (21h)، نهايته (7h)

🌱 بيانات البذر — ما تم ملؤه مسبقاً

"البذر" يعني بيانات أولية جاهزة قبل ما يبدأ أي مستخدم يستعمل النظام. بحال إعداد قاعدة عمل جاهزة.
