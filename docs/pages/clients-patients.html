🏢 صفحات العملاء (Clients)

📋 /dashboard/clients
Table تفاعلي: عرض كل العملاء مع عدد المرضى والخدمات والفواتير
بحث: بالاسم أو الهاتف أو البريد الإلكتروني
فلتر: أزرار لتصفية حسب النوع (Particulier · Entreprise · Assureur)
Pagination: 10 عناصر بالصفحة مع أزرار التنقل
نقر على الصف: ينقلك لصفحة التفاصيل
📄 /dashboard/clients/[id]
Card معلومات: هاتف · بريد · عنوان · ملاحظات · تاريخ الإنشاء
إحصائيات: عدد المرضى · الخدمات · الفواتير
قائمة المرضى: كل المرضى المرتبطين مع رابط لتفاصيل كل واحد
أزرار: تعديل · حذف (مع Modal تأكيد)
✏️ /dashboard/clients/new + /[id]/edit
فورم مشترك: ClientForm يُستخدم للإنشاء والتعديل
حقول: اسم · نوع (Select) · هاتف · بريد · عنوان · ملاحظات
Validation: Zod — أخطاء تظهر تحت كل حقل
Server Action: createClientAction / updateClientAction + revalidatePath

Table تفاعلي: اسم المريض · هاتف · تاريخ الميلاد · العميل المرتبط · عدد الخدمات
بحث: بالاسم أو الهاتف أو اسم العميل
Pagination: 10 عناصر بالصفحة
نقر على الصف: ينقلك لصفحة التفاصيل
📄 /dashboard/patients/[id]
Card معلومات: تاريخ الميلاد · هاتف · عنوان · ملاحظات طبية
ربط بالعميل: رابط مباشر لصفحة العميل مع Badge نوعه
إحصائيات: عدد الخدمات · هاتف العميل
أزرار: تعديل · حذف (مع Modal تأكيد)
✏️ /dashboard/patients/new + /[id]/edit
فورم مشترك: PatientForm يُستخدم للإنشاء والتعديل
حقول: عميل (Select dropdown) · اسم كامل · تاريخ الميلاد · هاتف · عنوان · ملاحظات طبية
ذكي: ?clientId=5 في URL يملأ العميل تلقائياً
Server Action: createPatientAction / updatePatientAction + revalidatePath

📊 لوحة التحكم (Dashboard)

✅ 5 بطاقات إحصائيات — عدد العملاء · المرضى · الخدمات · الفواتير · الدفعات (مباشرة من DB)
✅ أزرار سريعة — "Nouveau client" و "Nouveau patient" للوصول السريع
✅ Server Component — البيانات تُجلب من قاعدة البيانات عند كل زيارة

🏗️ النمط المعماري

📌 Server Components — كل page.tsx هو server component يجلب البيانات
📌 Client Components — الجداول التفاعلية والفورمات في _components/
📌 Server Actions — "use server" + revalidatePath لكل عملية CRUD
📌 Zod Validation — كل فورم يمر عبر schema validation قبل الحفظ
📌 Design System فقط — لا مكون واحد خارج النظام (Button, Input, Select, Table, Card, Badge, Modal)
📌 فورم مشترك — نفس الفورم للإنشاء والتعديل (mode: "create" | "edit")
