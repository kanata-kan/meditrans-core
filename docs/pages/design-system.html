🎯 التوكنات المركزية (Design Tokens)

ما هي التوكنات؟ قيم مركزية (ألوان، مسافات، ظلال) تُعرَّف مرة واحدة وتُستخدم في كل مكان.

بدلاً من كتابة #DC2626 في كل ملف، نكتب brand-primary ونغيرها من مكان واحد.

🔴 ألوان Brand
brand-primary: #DC2626 ← الأحمر الأساسي brand-hover: #B91C1C ← عند التمرير brand-soft: #FEE2E2 ← خلفية ناعمة
🟢 ألوان Status
status-success: #16A34A ← نجاح / مدفوع status-warning: #D97706 ← تحذير / معلّق status-danger: #DC2626 ← خطر / ملغى status-info: #2563EB ← معلومات / جزئي
🔵 ألوان Surface
surface-dark: #111827 ← Sidebar surface: #FFFFFF ← الأساسي surface-light: #F9FAFB ← خلفية الصفحة surface-muted: #F3F4F6 ← عناصر خامدة
🟡 ألوان Text
text-primary: #111827 ← النص الأساسي text-secondary: #4B5563 ← النص الثانوي text-muted: #9CA3AF ← تلميحات text-inverse: #FFFFFF ← نص على خلفية داكنة

🧱 المكونات الأساسية

كل مكون مكتوب بـ TypeScript strict، يستخدم forwardRef، ويقبل className إضافي. لا يحتوي أي مكون على منطق أعمال — UI فقط.

🔘 Button.tsx
Variants: primary · secondary · danger · ghost
Sizes: sm (h-8) · md (h-9) · lg (h-10)
States: loading (يعرض Spinner) · disabled (شفافية 50%)
📝 Input.tsx
Props: label · error · helperText · startIcon · endIcon
خاصية: focus ring أحمر (brand) · ring أحمر خطأ عند error
يدعم: كل أنواع HTML input (text, email, password, number)
📊 Table.tsx
Typed: Column<T> مع render function اختيارية
Pagination: أزرار previous/next مع عدد الصفحات
States: loading (skeleton rows) · empty (رسالة مخصصة)
قابل للنقر: onRowClick مع hover state
💬 Modal.tsx
Controlled: isOpen + onClose
Portal: يُرسم في document.body (لا z-index conflicts)
Accessibility: Escape key · overlay click · aria-modal
Sizes: sm (max-w-sm) · md (max-w-lg) · lg (max-w-2xl)
🏷️ Badge.tsx + statusToBadgeVariant()
Variants: default · success · warning · danger · info
Helper: statusToBadgeVariant("paid", "invoice") → "success"
يدعم: InvoiceStatus + ServiceStatus من constants.ts

🚫 قواعد صارمة

❌ لا ألوان hardcoded — كل شيء عبر Tailwind tokens
❌ لا منطق أعمال — المكونات تستقبل بيانات فقط
❌ لا تكرار — مكون واحد لكل غرض
❌ لا any — TypeScript strict بالكامل
✅ كل مكون يقبل className — للتخصيص من الخارج
✅ forwardRef على كل عنصر input
✅ cn() لدمج الأصناف (clsx + tailwind-merge)
